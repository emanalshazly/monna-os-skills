# ISO 42001 Mapping

## Reference: ISO/IEC 42001:2023 — Artificial Intelligence Management System

This document maps the AI Governance Framework Skill content to ISO/IEC 42001:2023 clauses. ISO 42001 provides requirements for establishing, implementing, maintaining, and continually improving an Artificial Intelligence Management System (AIMS).

---

## Clause 4: Context of the Organization

| Skill Content | ISO 42001 Clause | Mapping Rationale |
|---------------|------------------|-------------------|
| Phase 1: Scoping & Classification | 4.1: Understanding the organization and its context | Scope statement and system inventory capture internal and external context |
| Phase 1: Regulatory Classification | 4.2: Understanding the needs and expectations of interested parties | Regulatory requirements are a key interested party expectation |
| Phase 1: Stakeholder Mapping | 4.2: Interested parties include "partners, owners, providers, AI system operators, and users" | Stakeholder mapping explicitly identifies these roles |
| Phase 1: Scope Statement | 4.3: Determining the scope of the AIMS | Scope statement document defines AIMS boundaries |
| Core Principle 4: Stakeholder-Inclusive Design | 4.4: AI management system | Stakeholder inclusion is a foundational requirement for the AIMS |
